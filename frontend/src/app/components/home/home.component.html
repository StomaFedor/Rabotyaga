Краткое описание

Rabotyaga - это веб приложение по сбору и анализу резюме выпускиников МГТУ им Н.Е.Баумана на биржах труда

Приложение автоматически собирает резюме в базу данных и через веб-интерфейс выдает статистистичесие данные по резюме по запросу пользователя

Как источник данных был выбран сайт HH.ru, так как на нем доступно больше всего бесплатных резюме

Для получения данных используется парсинг веб-страниц
